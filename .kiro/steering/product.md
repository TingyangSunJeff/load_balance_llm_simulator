# Product Overview

This is an LLM Inference Simulator - a MATLAB-based research tool for simulating and benchmarking distributed inference of Large Language Models (LLMs) across various network and hardware configurations.

## Core Purpose
- Simulate distributed LLM inference across geographically distributed servers
- Benchmark different block placement, routing, and scheduling algorithms
- Analyze throughput, latency, and resource utilization in distributed LLM systems
- Support research on optimizing resource allocation for distributed AI inference

## Key Algorithms
- **Petals**: Baseline distributed inference approach
- **CG-BPRR**: Proposed heuristic algorithm (Column Generation Block Placement and Request Routing)
- **ILP/LP-based methods**: Integer Linear Programming and Linear Programming relaxation approaches
- **Greedy algorithms**: Various heuristic approaches for comparison

## Research Context
This simulator supports academic research on distributed AI systems, specifically focusing on optimizing resource allocation for geographically-distributed LLM inference. The work is published in IFIP WG 7.3 Performance 2025 conference.
